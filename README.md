# Учебный транслятор конфигурационного языка в XML

## Назначение

Программа выполняет трансляцию учебного конфигурационного языка в формат XML.
Входной текст читается из стандартного ввода, результат выводится в стандартный вывод.

## Поддерживаемые конструкции

- Числа вещественного типа
- Массивы array(...)
- Словари { ключ : значение }
- Объявление констант: const имя = значение
- Использование констант: .[имя].
- Многострочные комментарии: #= ... =#

## Использование

python main.py < config.txt

## Примеры

В архиве представлены три примера конфигураций из разных предметных областей:

- economy.txt — экономика
- it.txt — информационные технологии
